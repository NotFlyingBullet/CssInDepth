Глава 11 Фоны тени и режимы смешивания.


11.1 Градиенты

	background-image - устанавливает изображение или градиентный фон
	background-position - начальное положение фонового изображения
	background-size - размеры внутри элемента
	background-repeat - повторение изображения для заполнения элемента
	background-origin - положение изображения относительно границ блока элемента
	background-attachment - будет ли фоновое изображение перемещаться вместе с элементом или будет неподвижным
	background-color - фон позади изображения

	linear-gradient - принимает right top.. rad turn deg rgb #

	Можно выполнять градиенты с большим кол-м цветов, каждый из кот называется цветовым узлом.

	Примеры градиентов
	https://css-tricks.com/stripes-css/
	https://developer.mozilla.org/ru/docs/Web/CSS/radial-gradient()				

	radial-gradient - Повторяет пропорции элемента можно использовать узлы.
	Можно сдклать градиент круглым и задать точку центра градиента.
	Можно задать несколько цветов и определить их положение с помощю процентов и единиц измерения длинны.

	Вместо перехода от белого к черному, лучше применить белый и светлосерый, или переход между двумя оттенками синего.


11.2 Тени

	box-shadow: x y радиус_размытия радиус_распространения black;
	радиус_распространения - размер тени.
	радиус_размытия - края тени.

	box-shadow: inset 0 0 0.5em #124;//Внутрянняя тень


11.3. Режимы смешивания стр. 342

	background-image может принимать много аргументов разделенных запятыми.
	Первое изображение будет перекрывать последующие.

	background-size		//Также могут принимать насколько значений.
	background-repeat //Также могут принимать насколько значений.

	cover - заполнить.
	contain - уместить целиком с сохранением пропорций (принимает длинну и ширину).

	background-blend-mode://Режим смешивания принимает
	multiply
	color-burn
	difference
	и др.


11.3.3. Добавление текстуры изображению.

	Если не нужно менять оттенок снимка, то нужно использовать черно-белое изображение текстуры.

	Текстуру плучают одним из режимов смешивания: overlay, hard-light, soft-light.

	Режим soft-light намного лучше подходит для темных изображений текстуры, тогда как hard-light overlay предпочтительно использовать для более темных изображений.


11.3.4. Микширование режимов смешивания.

	Использование	background-blend-mode: ограничевается цветами и изображениями одного элемента.

	Mix-blend-mode: - обеспечивает наложение нескольких элементов.


	Итоги главы:
	- Использовать градиенты и тени, чтобы добавить глубины;
	- Применять градиенты с цветовыми узлами, что бы создать эффект полос;
	- Градиент всегда лучше чем простой фон;
	- Использовать режимы смешивания, чтобы придать изображениям необходимый оттенок или наложить текстуру.

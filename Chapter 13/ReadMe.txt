	@font-face - набор правил css определяет какие шрифты подключить.

	формат шрифтов WOFF2 и более старый WOFF.(Web open font format)

	@font-face {
  font-family: 'Roboto';
  font-style: normal;
  font-weight: 300;
  font-display: swap;
  src: local("Roboto-Light"), local("Roboto-Light"),
  	url(https://fonts.gstatic.com/s/roboto/woff2) format('woff2');//Будет использоваться первым
  	url(https://fonts.gstatic.com/s/roboto/woff) format('woff');//Если браузер не поддерживает WOFF2
  unicode-range: U+0460-052F, U+1C80-1C88, U+20B4, U+2DE0-2DFF, U+A640-A69F, U+FE2E-FE2F;
	}

	line-hight: normal;//1.2em
	leter-spacing: 0.01em//0.01-0.03em оптимально

	Идеальное значение для основного текста line-hight 1.4 - 1.6 em.

	Для длинных строк следует делать большую высоту строки.
	В идеале стоит делать строки длинной 45-75 символов.

	Межстрочный интервал называется интерлиньяж, а интервал между буквами - трекингом.
	Чтобы перевести пункты в пиксели надо умножить значение в пунктах на 1.333(на 1 дюйм приходится 96 пикселов или 72 пункта 96/72=1,333),	это число следует разделить на размер шрифта, чтобы получить значение в безразмерных единицах.

	Вертикальный ритм - соблюдение сетки базовых горизонтальных линий кратных базоовой высоте строки.

	Вспышки нестилизованного текста FOUT (flash of unstyled text).
	Вспышки невидимово текста FOIT (flash of invisible text).

	Библиотека Font Face Observer позволяет выбрать вид вспышки.
	fontfaceobserver.com

	FOIT - лучше использовать при быстром соединении.
	FOUT - при медленном соединении.

	Новое свойство css font-display позволяет контролировать загрузку шрифтов.

	@font-face{
	font-family:"Roboto";
	font-style: normal;
	font-weight: 300;
	src: local("Roboto Light"), local("Roboto Light"),
	url(http://exemple.com) format("woff2"),
	url(http://exemple.com) format("woff");
	font-display: swap;
	}

	auto - По умолчанию (вспышка невидимого текта).
	swap - Отображает системные шрифты, а web шрифты после загрузки.
	fallback - Короткое время 100мс текст невидим, если шрифты еще не загрузились то отобразятся системные, а после полной загрузки web шрифты.
	optional - Браузер сам решает какие шрифты использовать. Обычно при медленном соединении даже не будет попытки загрузить системные шрифты.

	Для быстрых соединений лучше использовать fallback, а для медленных swap.

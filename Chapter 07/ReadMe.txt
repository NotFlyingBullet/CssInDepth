Глава 7 Контексты позиционирования и наложения.

	position: static;// Статическое позиционирование (Не позиционированный элемент)

	Для правильного отображения в разных браузерах необходимо вместе с position указывать top left right bottom. 


7.1. Фиксированное позиционирование.

	position: fixed;// Извлекается из потока 
									// Произвольное позиционирование в области просмотра.

	top right bottom left// Как далеко д находиться элемент от сторон области просмотра
											// Установив эти значения неявно задаются и размеры элемента

	Можно использовать свойства width и hight

	Элемент станивотся строчно-блочным (width 100%)


7.2. Абсолютное позиционирование.

	position: absolute;//Извлекается из потока

	top right bottom left// Как далеко д находиться элемент от сторон области просмотра
											// Установив эти значения неявно задаются и размеры элемента

	Работает не относительно области просмотра, а относительно содержащего элемена(ближайшего позиционированного родительского элемента).

	Если не один из предков не позиционирован, то элемен позиционируется относительно области равной области просмотра привязанная к верхней части страницы.

	Элемент станивотся строчно-блочным (width 100%)
	

7.3. Относительное позиционирование.
	
	Не извлекается из потока

	position: relative;

	top right bottom left// Как далеко д находиться элемент от сторон области просмотра
	
	css-trick.com/examples/shapesofcss//Фигуры построенные с помощю css.						


7.4.3. Контексты наложения.

	Все элементы располагаются в следующем порядке от заднего плана к переднему:
	- Корневой элемент контекста наложения;
	- Позиц-е элементы со значением z-index <0 и их дочперние элементы;
	- Не позиц-е элементы;
	- Позиц-е элементы со свойством z-index: auto; и их дочперние элементы;
	- Позиц-е элементы со значением z-index: >0 и их дочперние элементы.

		
7.5.Липкое позиционирование.

	position: sticky;
	top: 1em;//Будет пристыковано на расстояние 1em от верхнего края.


Итоги главы:
	
	- Использовать фиксированное позиц-е для модальных окон;
	- Использовать абсолютное позиционирование для всплывающих окон, раскрывающегося меню, и прочих динамических взаимодействий;
	- Помнить о проблеммах совместимости данных типов позиц-я;
	- У свойства z-index есть две особенности: оно работает только с позиционированными элементами, с его помощю создается новый контекст наложения;
	- Помнить о потенциальных ошибках при создании нескольких контекстов наложения на странице;
	- Отслеживать реализацию в браузерах поддержки Sticky позиц-я.
